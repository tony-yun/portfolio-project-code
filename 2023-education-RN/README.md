## 📌 주요 기능 (Features)

### 1. 백그라운드 재생 및 다운로드 설정

- **백그라운드 재생 지원**

  - 학습 콘텐츠를 앱을 벗어나도 계속 들을 수 있도록 설정 가능

- **건너뛰기 간격 조정**

  - 사용자가 원하는 단위(예: 10초)로 설정

- **SD 카드 저장 기능**
  - SD 카드가 연결되어 있을 경우 다운로드 위치를 SD 카드로 변경 가능
  - 총 용량 및 사용 용량 확인 가능

### 2. 결제 기능

- **결제 수단 선택 가능**
  - 신용카드/체크카드, 무통장 입금, 실시간 계좌이체, 평생교육바우처 지원
- **예상 적립 포인트 표시**
- **개인정보 수집 및 결제 동의 안내 포함**
